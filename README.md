# 🚇 Metro Rush 3D

Бесконечный 3D-раннер для браузера в духе классических subway-раннеров: беги по рельсам,
уворачивайся от поездов, собирай монеты. Оптимизирован под мобильные устройства и плавные
**120 fps** (рендер идёт на частоте дисплея через `requestAnimationFrame`).

**Играть:** https://yokki-vans.github.io/metro-rush-3d/

## Особенности

- 🏃 Три полосы, свайпы: ←/→ — смена полосы, ↑ — прыжок, ↓ — кувырок (в воздухе — быстрое приземление)
- 🚆 Стоячие и движущиеся поезда (с гудком и фарами), барьеры, рамки «проползи под знаком»
- 🛤 Рампы: взбегай на крыши вагонов и собирай монеты наверху — по крышам движущихся поездов тоже можно бежать (но крыша «уезжает» из-под ног)
- 🌗 Полный цикл дня и ночи: рассвет, полдень, закат, звёзды, луна, светящиеся окна
- 🌧 Погода: ясно / дождь с молниями и громом / туман / снег — плавные переходы
- 🏙 Меняющиеся биомы: даунтаун, старый город, парк, промзона с контейнерами и тоннели
- 🌀 Мир радиально изогнут (vertex-шейдер): трасса плавно «уходит за угол», геймплей при этом честно прямой
- 🚀 Бустеры: 🧲 магнит, 👟 супер-прыжок (перепрыгивай поезда!), ×2 очки, 🛡 щит со «спасительным прыжком»
- 🤖 6 скинов робота — открываются за накопленные монеты (копилка живёт между забегами)
- 🏆 Локальная таблица лидеров топ-10 с именем игрока
- 💡 Свечение: текстурные монеты с пульсирующим ореолом, гало фонарей и ламп тоннеля, облака, виньетка, FOV-кик от скорости
- 🔊 Звук полностью синтезируется WebAudio на лету: музыка-секвенсор, эффекты, дождь, гром, гудки — ни одного аудиофайла
- ✨ Эффекты: частицы (монеты, пыль, конфетти, краш), спидлайны, тряска камеры, слоумо при столкновении
- 📱 Тач-управление + клавиатура (стрелки/WASD, Space, P — пауза, M — звук)
- ⚙️ Автогувернёр пиксель-рейтио: на слабых устройствах разрешение плавно снижается ради fps

## Управление

| Действие | Тач | Клавиатура |
|---|---|---|
| Сменить полосу | свайп ←/→ | ←/→ или A/D |
| Прыжок | свайп ↑ или тап | ↑ / W / Space |
| Кувырок | свайп ↓ | ↓ / S |
| Пауза | кнопка II | P |
| Звук | кнопка 🔊 | M |

## Запуск локально

Любой статический сервер, например:

```bash
python3 -m http.server 8013
# открой http://localhost:8013
```

`?ghost=1` в URL — режим полёта без столкновений (для отладки).

## Технологии

Чистый three.js без сборщика: ES-модули + import map, всё хостится статически (GitHub Pages).
Геометрия чанков мира слита в 2–3 меша на чанк (включая запечённые вершинные цвета и UV-атлас),
поезда/препятствия/монеты — `InstancedMesh`-пулы: суммарно ~30 draw-calls за кадр.
Все текстуры рисуются в canvas при старте.
